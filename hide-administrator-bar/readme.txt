Use this plugin to Hide Administrator bar for Admin and Non Admin users.

== Description ==
 
Using this plugin users can Hide admin bar for all users or only for non-admin users or also including admin in front-end.

= Details =

* Can Hide Admin Bar for all Users in Frontend.
* Can Hide Admin Bar for Non Admin users in Frontend.
* Can Hide Admin Bar for Admin users in Frontend.
* Setting available in Admin section for configure who can see and who cannot see the admin bar.

== Installation ==

The Plugin can be installed in two ways.

= Download and Install =

1. Download and upload the plugin files to the `/wp-content/plugins/hide-administrator-bar` directory from the Wordpress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress admin.

= Using Plugin Interface =

1. Open the plugin Interface in the wordpress admin and click on "Add new".
2. In the Search box enter "Hide Administrator Bar" and hit Enter.
3. Click on "Install" to install the plugin.
